### üåç Travel Memory Globe
**Interactive 3D globe for organizing travel photos by country**

An immersive React application that lets you relive your travel memories by clicking on country markers on a beautiful 3D rotating globe. Upload photos, create albums, and visualize your adventures around the world.

**Tech Stack:** React, Three.js, TypeScript, Vite  
**Features:**
- Interactive 3D globe with drag-to-rotate controls
- 20+ pre-loaded country destinations
- Photo album system with gallery navigation
- Travel statistics dashboard
- Space-themed UI with smooth animations
- Session storage for photo persistence

---

### üí∞ Budget Tracker
**Full-stack personal finance management application**

A comprehensive financial tracking system built with Flask that helps you manage your money across multiple accounts, set budgets, and track your spending habits with beautiful visualizations.

**Tech Stack:** Flask (Python), SQLite, JavaScript, Matplotlib  
**Features:**
- Multi-currency support (USD, EUR, GBP, JPY, CAD, AUD)
- Multiple account types (checking, savings, credit, investment)
- Budget limits with visual progress tracking
- Transaction tagging and categorization
- CSV export functionality
- Secure user authentication
- Real-time dashboard analytics

---

### üó°Ô∏è Quest Board - Gamified Task Manager
**Turn your to-do list into an epic RPG adventure**

A unique productivity app that transforms mundane tasks into exciting quests. Level up your hero, earn coins, build streaks, and climb through 16 hero ranks as you complete your daily objectives.

**Tech Stack:** Vanilla HTML/CSS/JavaScript  
**Features:**
- 16 hero ranks from Squire to Hero of the Universe
- 4-tier difficulty system (Common, Rare, Epic, Legendary)
- XP and coin reward system
- 5-task streak bonuses
- Time period organization (Today, Week, Month, Someday)
- Confetti celebrations on level-up
- Beautiful glassmorphic UI with animated starfield
- LocalStorage persistence
