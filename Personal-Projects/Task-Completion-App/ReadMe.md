## How to Use

1. **Open the File**: Simply open `Quest-Board.html` in any modern web browser
2. **Create a Quest**: 
   - Type your task in the input field
   - Select a difficulty rank (Common/Rare/Epic/Legendary)
   - Choose a time period
   - Click the + button or press Enter
3. **Complete Quests**: Click the circle button next to any quest to mark it complete
4. **Track Progress**: Watch your XP bar fill, level up, and earn your hero rank
5. **Build Streaks**: Complete 5 quests in a row for bonus coins

## Technical Details

- **No Dependencies**: Pure HTML, CSS, and JavaScript
- **Local Storage**: All progress automatically saved to browser
- **No Server Required**: Works completely offline
- **Browser Compatibility**: Works in all modern browsers (Chrome, Firefox, Safari, Edge)

## Data Persistence

All your data is stored locally in your browser using localStorage:
- Tasks and completion status
- XP, level, and coins
- Current streak count

**Note**: Clearing browser data will reset your progress.

## Installation

No installation needed! Just download and open the HTML file in your browser.

## Tips for Maximum Productivity

1. **Start Small**: Begin with Common quests to build momentum
2. **Be Realistic**: Match difficulty to actual task complexity
3. **Use Time Periods**: Organize by urgency (Today) vs. importance (Someday)
4. **Maintain Streaks**: Complete at least one quest daily to keep your streak alive
5. **Celebrate Wins**: Watch for the confetti when you level up!
